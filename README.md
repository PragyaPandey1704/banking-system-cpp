# 💰 Banking System in C++

A console-based Banking System developed in C++ using Object-Oriented Programming (OOP) concepts. This project simulates basic banking operations such as account creation, deposit, withdrawal, and balance inquiry.

---

## 🚀 Features

- Create new bank account (Savings / Current)
- Support for single and joint accounts
- Deposit money
- Withdraw money
- Balance inquiry
- Change PIN securely
- Close account
- Admin panel to view all accounts
- PIN-based authentication for security

---

## 🛠️ Technologies Used

- C++
- STL (Vector)
- Object-Oriented Programming (OOP)
